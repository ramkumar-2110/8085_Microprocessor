##8085 Microprocessor

#features
    *it is a 8 bit basic Microprocessor.
    *it have von Neumann architecture.
    *it have 16 address lines and 8 data lines in a multiplexed form.
    *Minimum clock frequency is 500Hz and Maximum is 3MHz.
    *it is a 40 pin IC.
    *it uses single memory for to store program and data.
    *it have 6 general purpose register.
